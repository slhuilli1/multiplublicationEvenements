Bienvenue sur mon projet de multi-publication d’événements, principalement culturels, proposé via ce service.

Le principe de cet outil est volontairement simple et peut se résumer en une seule phrase : publier un événement — qu’il soit culturel, associatif, festif, coquin ou de toute autre nature — en un seul clic.

Concrètement, il vous suffit de décrire votre événement sur sa page web en respectant une structure normalisée, basée sur le standard défini par schema.org. Cette description inclut toutes les informations essentielles : nature de l’événement, date, lieu, organisateur, etc.

Une fois votre page correctement structurée selon cette norme, mon système se charge automatiquement de collecter les informations de l’événement afin de les rediffuser vers les plateformes et organisations susceptibles d’en assurer la promotion.
L’objectif est de simplifier la diffusion, d’élargir la visibilité de vos événements et de centraliser leur promotion sans effort supplémentaire de votre part.
